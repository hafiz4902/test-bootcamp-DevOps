      5.Infrastructure as a Code (IAAC)

   persamaannya yaitu sama samauntuk mengolah container, sedangkan perbedannya sebagai berikut:
   - Teknologi yang terlibat
   - Arsitektur
   - Load balancing
   - skalabilitas
